# 子模板合集

## 1. 人物出场片模板
【风格】真人电影级实拍结合高端 CGI，[画幅]，[时长]秒，整体为[审美体系]。
【主体】一位[人物类型]，外形为[发型/服装/配饰]，气质[关键词]。
【场景】位于[空间]，背景有[环境元素]。
【时间轴】
- 0-3秒：人物亮相，建立世界和气场。
- 3-6秒：人物做出第一个高识别动作。
- 6-10秒：动作升级，加入镜头翻转/特效交互。
- 10-13秒：进入高潮，人物与环境或能量发生强交互。
- 13-15秒：最强姿态 + LOGO/标题封板。
【记忆点】人物的[标志动作/表情/道具]定格。

## 2. 产品广告片模板
【风格】高端广告质感，[画幅]，[时长]秒，风格偏[奢华 / 科技 / 极简 / 未来]。
【主体】主体为[产品名称]，材质为[玻璃 / 金属 / 液态 / 哑光]。
【场景】场景为[实验室 / 黑棚 / 水面 / 镜面空间 / 未来装置空间]。
【时间轴】
- 0-2秒：从局部细节建立材质和神秘感。
- 2-5秒：产品主体逐渐显现。
- 5-8秒：展示结构、功能、材质细节。
- 8-12秒：特效或空间升级，突出价值感。
- 12-15秒：产品完整亮相，品牌名或 slogan 封板。
【记忆点】产品以最有辨识度的角度和光线定格出现。

## 3. 品牌 LOGO 片模板
【风格】短促、直接、强记忆点，[画幅]，[时长]秒。
【主体】核心为[品牌名 / LOGO / 字体装置]。
【时间轴】
- 0-2秒：建立氛围与前奏。
- 2-5秒：LOGO 前的符号元素逐渐聚集。
- 5-8秒：通过能量、粒子、机械、液态或光束形成视觉爆发。
- 8-10秒：LOGO 完整出现并短暂停留。
【记忆点】LOGO 与光效、粒子、声音共同形成封板。

## 4. 氛围预告片模板
【风格】电影预告感，[画幅]，[时长]秒，强调悬疑、史诗或压迫气氛。
【主体】主体为[角色/事件/地点/灾难/神秘物体]。
【时间轴】
- 前段：建立世界观与不安感。
- 中段：放出一个关键动作或危险信号。
- 后段：快速升级为视觉高潮。
- 结尾：留下一句标题、一个符号或一帧极强画面作为悬念。
【记忆点】让观众记住“那个瞬间”，而不是所有镜头平均发力。
