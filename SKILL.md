# 高质感视频提示词万能骨架

## 适用时机
当用户要写视频提示词，并且希望结果更像成片而不是松散的形容词堆砌时，使用这套方法。尤其适合：
- 人物出场片
- 品牌广告片
- 产品质感片
- 时尚影像
- 电影感短预告
- 科幻、奇幻、赛博朋克、悬疑风视频

不优先用于：
- 生活流 vlog
- 纯对白剧情
- 纪录片式写实片段
- 追求随意自然感的镜头

## 核心原则
高质量视频提示词，不是把词写华丽，而是把审美变成可执行的镜头系统。

始终遵守这些规则：
1. 先定片种，再写画面。
2. 少写抽象赞美词，多写镜头、动作、光线、材质、声音。
3. 主线只能有一个，所有细节都服务同一个主视觉中心。
4. 视频最重要的是节奏推进，不是单帧美堆砌。
5. 特效必须和现实材质、反光、阴影、环境亮度产生交互。
6. 结尾必须有高潮封板或明确记忆点。
7. 信息不够时，先问再写，不要擅自补全关键商业信息。
8. 最终输出给用户的提示词正文总长度不能超过 2000 字。

## 先做一句话主线
先把需求压缩成一句：
“谁，在什么地方，用什么动作，制造什么高潮。”

如果用户给的内容太散，先帮他提炼出这条主线，再展开。

## 先判断信息是否足够
在动笔之前，先检查是否缺少高影响信息。以下项目里，只要缺 2 项以上，或缺了明显关键项，就先与用户交互补信息，再输出正式提示词。

高影响信息优先级如下：
1. 片种/用途：品牌广告、人物出场、产品片、情绪短片、MV、预告片等。
2. 主体：谁或什么是主角，人物/产品/品牌分别是什么。
3. 场景：发生在哪里，现实空间还是虚构空间。
4. 风格方向：赛博朋克、黑金奢华、写实电影、复古、未来科技等。
5. 画幅与时长：16:9、9:16、1:1；时长几秒。
6. 品牌露出：是否需要展示品牌 LOGO、品牌名、口号、字幕、片尾封板。
7. 输出用途：是给模型直接生成，还是给导演、剪辑、拍摄团队参考。
8. 声音需求：是否需要音乐、音效、节奏卡点、无对白等。

## 缺信息时的交互规则
当信息不足时，不要直接写长提示词，先用简洁的方式向用户追问。追问规则：
- 每轮优先只问最关键的 3 到 6 项，避免一次问太散。
- 问题要具体、可直接回答，尽量做成填空或选项。
- 优先问会显著影响成片方向的内容。
- 对品牌类需求，默认必须确认是否需要品牌 LOGO、品牌名、口号、片尾封板。
- 如果用户只想快速出稿，可询问“是否允许我用默认值补全未指定信息”。只有用户明确允许时，才可代补默认值。

可直接复用下面这类追问格式：
1. 这是人物片、品牌片、产品片，还是情绪氛围短片？
2. 主角是谁，核心主体是什么？
3. 场景想放在什么地方？
4. 你更偏什么风格：写实电影、赛博朋克、奢华时尚、未来科技，还是别的？
5. 画幅和时长要多少？比如 16:9、15 秒 / 9:16、10 秒。
6. 是否需要展示品牌 LOGO、品牌名、标语或片尾封板？
7. 是否需要强节奏卡点、音乐和音效？

## 信息足够时再输出
当关键输入已经足够清晰时，再开始组织正式提示词。
如果用户提供的是零散信息，可先用 1 句话复述你理解的主线，再输出完整提示词。

## 标准输出顺序
输出视频提示词时，优先按这个顺序组织：
1. 风格
2. 主体
3. 场景
4. 镜头
5. 时间轴
6. 光线
7. 材质
8. 声音
9. 记忆点

## 每个模块该写什么
### 1) 风格
明确：
- 实拍、CG、还是实拍+CG
- 画幅比例
- 时长
- 分辨率
- 审美系统与总体气质

### 2) 主体
明确：
- 主角是谁
- 发型、服装、配饰
- 第一印象与气场
- 若是产品，则写清材质、形态、品牌调性

### 3) 场景
明确：
- 空间类型
- 背景可识别元素
- 主道具或环境装置
- 明暗关系与氛围

### 4) 镜头
明确：
- 推、拉、环绕、微距、仰拍、翻转、长镜头、快切等关键镜头语言
- 节奏如何从平稳进入升级与爆发

### 5) 时间轴
这是最关键部分。按秒写清：
- 这几秒看到什么
- 主体做什么
- 镜头怎么动
- 情绪如何升级

默认建议：
- 起：建立世界与主体
- 承：动作推进
- 转：能量、镜头或情绪升级
- 合：高潮封板

### 6) 光线
明确：
- 主色与辅色
- 主光、背光、轮廓光
- 特效光颜色与层次
- 反光、阴影、动态环境光

### 7) 材质
明确：
- 哪些质感必须真实：皮肤、金属、布料、玻璃、水面、机械等
- 反射、折射、阴影、动态模糊要符合物理逻辑
- 特效不应像贴纸，而要嵌入场景

### 8) 声音
即使用户主要写画面，也可补充：
- 背景音乐风格
- 动作音效
- 环境音
- 特效能量音
- 呼吸、摩擦、风声、金属声等细节音

### 9) 记忆点
必须明确最后让观众记住哪一帧：
- 一个高识别动作
- 一个封神姿态
- 一个 LOGO/标题/产品定格画面

## 输出长度控制
最终给用户的正式提示词正文必须控制在 2000 字以内。

压缩顺序如下：
1. 先删重复修饰词和同义形容词。
2. 再压缩声音、材质、光线中的次要描述。
3. 保留时间轴、镜头、主体、场景、高潮记忆点这些核心信息。
4. 如果仍超长，优先输出“精简完整版”，而不是牺牲主线和节奏。

不要把分析过程、原理解释、冗长说明混进最终提示词正文里。

## 写作方法
### 快速版
当用户只要速写或想先搭框架时：
- 先给一句话主线
- 再给 4 段式节奏：起、承、转、合
- 最后补 3 个关键词：光线、材质、记忆点
- 仍需遵守 2000 字上限

### 完整版
当用户要成片级提示词时：
- 严格按 9 模块输出
- 时间轴必须具体到秒段
- 让镜头、动作、光线、材质、声音彼此配合
- 仍需遵守 2000 字上限

## 改写与优化规则
当用户给了一个已有提示词，优先做这些优化：
1. 删掉不服务主线的酷元素。
2. 把“高级、震撼、未来感”改成可执行画面信息。
3. 给缺失的节奏补上起承转合。
4. 把高潮从“整体都很满”改成“最后一击最强”。
5. 强化真实材质与特效交互。
6. 把结尾改成更可传播的封板画面。
7. 若原提示词缺少品牌露出方式，先确认是否需要 LOGO / 标题 / 标语。

## 输出时的语气
- 直接、专业、可执行。
- 不空喊“高级感”。
- 少解释理念，多给可落地文本。
- 默认输出中文；若用户要求英文或双语，再切换。

## 自检清单
输出前检查：
- 是否只有一个主线
- 是否有明确时间轴
- 是否有镜头语言
- 是否有光线与材质逻辑
- 是否有高潮记忆点
- 是否把抽象词翻译成了执行信息
- 信息不足时是否先追问
- 是否确认了品牌 LOGO / 品牌名 / 片尾封板需求
- 正式提示词正文是否不超过 2000 字

## 简版骨架
可直接按下面格式写：

【风格】
[片种 / 画幅 / 时长 / 质感 / 审美体系]

【主体】
[谁 / 外观 / 服装 / 配饰 / 气质]

【场景】
[在哪里 / 背景元素 / 主道具 / 氛围]

【镜头】
[推进 / 拉远 / 环绕 / 微距 / 仰拍 / 翻转 / 节奏变化]

【时间轴】
0-[x]秒：[建立世界与人物]
[x]-[x]秒：[主体动作推进]
[x]-[x]秒：[镜头或能量升级]
[x]-[x]秒：[高潮记忆点]
最后1秒：[LOGO / 标题 / 黑场残像 / 收尾]

【光线】
[主色 / 辅色 / 主光 / 背光 / 特效光 / 反射关系]

【材质】
[皮肤 / 金属 / 布料 / 玻璃 / 水 / 机械等真实表现]

【声音】
[音乐 / 动作音 / 环境音 / 特效音 / 收尾]

【记忆点】
[最终最强一帧是什么]
