# 质检清单

写完提示词后，逐项检查：

## 结构
- [ ] 我已经先定片种，而不是直接堆形容词
- [ ] 主线只有一个
- [ ] 提示词有清晰的起承转合
- [ ] 我能指出最强高潮在哪一秒
- [ ] 我能指出最后一帧记忆点是什么

## 镜头
- [ ] 不只是写“画面内容”，还写了镜头怎么动
- [ ] 至少说明了景别或角度变化
- [ ] 镜头节奏是递进的，不是平均发力

## 光线与色彩
- [ ] 主色和辅色明确
- [ ] 主光、背光、轮廓光关系清楚
- [ ] 特效光与环境、人物反光有交互

## 材质与真实感
- [ ] 哪些材质必须真实已经写清楚
- [ ] 特效不是贴纸，而是会影响环境
- [ ] 反射、阴影、动势、质感有被强调

## 声音
- [ ] 至少有音乐风格
- [ ] 至少有关键动作音效
- [ ] 结尾有收束方式

## 减法
- [ ] 没有塞太多不服务主线的元素
- [ ] 没有只写“很酷、很炸、很高级”这种空词
- [ ] 每个关键词都能转化成执行画面
